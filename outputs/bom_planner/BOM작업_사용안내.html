BOM 구성 작업 사용안내

이 도구는 기존 ERP/DB에 직접 저장하지 않고, 담당자가 BOM 초안을 만든 뒤 검수자가 확인해 CSV로 제출하기 위한 독립 HTML입니다.

1. 시작

  1. bom_planner.html을 더블클릭합니다.
  2. 상단의 작업자명에 이름을 입력합니다.
  3. 왼쪽에서 상위 품목을 고릅니다.
  4. 오른쪽 하위 후보에서 ＋, 더블클릭, 드래그 앤 드롭, ERP 코드 직접 입력 중 하나로 하위 품목을 추가합니다.

2. 저장

  • 작업은 브라우저에 자동 저장됩니다.
  • 다른 PC로 옮기려면 반드시 작업파일 저장으로 JSON 파일을 내보내세요.
  • 작업을 이어갈 때는 작업파일 불러오기를 사용합니다.

3. 검수

  • 검수자 모드로 바꾸면 미완료 상위 품목까지 경고로 표시됩니다.
  • 오류는 확정 CSV 생성을 막습니다.
  • 경고만 남은 경우에는 검수자가 사유를 비고에 적고 확정 CSV를 만들 수 있습니다.

4. 제출

  • 확정 CSV: 최종 BOM 원장 반영용입니다.
  • 검수 CSV: 오류/경고 목록 전달용입니다.
  • CSV는 Excel에서 한글이 깨지지 않도록 UTF-8 BOM 형식으로 저장됩니다.
