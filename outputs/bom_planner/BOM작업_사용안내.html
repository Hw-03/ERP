BOM 작업 사용안내

컴퓨터에 익숙하지 않아도, 화면 안내대로 따라가면 끝납니다.

1처음 화면 (탭 3개)

  • bom_planner.html을 더블클릭해 엽니다.
  • 위쪽에 탭이 3개 있어요:
    • 📂 작업하기 — 부서를 골라 새로 BOM 만들기
    • ✓ 완료한 BOM — 끝낸 것 모아보기 + 한꺼번에 다운로드
    • ⏳ 아직 안 한 것 — 미처리 상위 품목 한눈에 보기

2부서 고르기

  • 「작업하기」 탭에서 자기 부서를 한 번 누릅니다 (튜브 / 고압 / 진공 / 튜닝 / 조립 / 출하).
  • 그 부서가 만드는 상위 품목 목록으로 넘어갑니다.

3상위 품목 고르기

  • 위 검색창에 품명이나 코드를 적으면 바로 걸러집니다.
  • BOM을 만들 품목을 한 번 누릅니다.
  • 오른쪽에 「아직 안 함 / 작업 중 / 끝남」 표시가 있어 진행 상태를 알 수 있어요.

4하위 부품 추가

  • 오른쪽 부품 목록에서 한 번 누르면 즉시 추가됩니다 (수량 1, 단위 EA 기본).
  • 왼쪽에 들어간 부품의 수량은 ＋ / − 버튼으로 조정합니다.
  • 「검수 끝남」 스위치를 켜면 그 줄이 검토 완료로 표시됩니다.
  • 비고를 적으려면 줄 오른쪽의 ✎ 아이콘을 누르세요.
  • 잘못 넣었다면 × 아이콘으로 뺄 수 있어요.
💡 팁 — 작업은 자동 저장됩니다. 브라우저를 닫았다 켜도 그대로 이어집니다.

5검토 + 완료 처리

  • 아래쪽 「다음: 검토하기 →」 버튼을 누릅니다.
  • 요약 카드에서 「잘된 점」과 「고쳐야 할 곳 / 확인할 곳」을 한눈에 봅니다.
  • 빨간 항목이 있으면 「← 더 작업하기」로 돌아가 고칩니다.
  • 이상 없으면 「✓ 이 BOM 완료 처리」를 누릅니다 — 이 BOM이 「완료한 BOM」 탭으로 모입니다. 이 시점에는 파일이 만들어지지 않습니다.

6전체 다운로드 (작업이 다 끝난 뒤)

  • 「✓ 완료한 BOM」 탭으로 이동합니다.
  • 위쪽 「📥 전체 다운로드 (CSV)」 버튼을 한 번 누르면 그동안 완료한 모든 BOM이 한 번에 CSV 2개로 다운로드됩니다.
    • bom_final_...csv: 모든 완료 BOM 자료
    • bom_review_...csv: 확인이 필요했던 항목 기록
  • 잘못 완료 처리한 게 있으면 그 줄의 「완료 취소」를 누르거나, 「다시 열기」로 편집을 이어갈 수 있습니다.
⚠ 완료 처리는 빨간 항목이 모두 사라진 뒤에만 됩니다. 노란 항목은 통과해도 완료 가능합니다.

7도움이 되는 단축키

  • Esc — 한 단계 뒤로 갑니다 (편집 → 상위 → 처음 화면).
  • Enter — 검색창에서 첫 결과로 이동.

?자주 보이는 메시지

  • "공정 순서가 어색해요" — 부품의 공정 단계가 상위와 비슷하거나 더 늦어 보일 때 뜹니다. 정말 맞다면 비고에 이유를 적어주세요.
  • "같은 부품이 두 번 들어가 있어요" — 같은 부품이 중복 추가됐어요. 한 줄만 남기고 나머지는 × 로 빼주세요.
  • "부품끼리 서로의 하위가 되어 있어요" — A→B→A 처럼 빙글 도는 관계입니다. 한쪽을 빼야 합니다.
