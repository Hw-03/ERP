그룹 A — 핵심 흐름 (전부 해야 함)

A1 원자재 입고 (결재 없음) ▼

  1. 입출고 → 원자재 입고
  2. 품목 검색: TR (튜브 원자재) 아무거나
  3. 낱개 / 수량 50 / 제출
ℹ️ 원자재 입고는 결재 없이 즉시 처리됨 — 부서 결재 문구가 뜨지 않는 것이 정상 흐름.
A2 창고 → 부서 낱개 (창고 결재) ▼
→ E15 김도영 (요청)
  1. 입출고 → 창고 반출
  2. A1에서 입고한 TR 품목 / 낱개 / 수량 30 / 제출
→ E22 이필욱 (승인)
  1. 승인함 → 해당 요청 → 승인
A3 창고 → 부서 BOM (창고 결재) ▼
⚠️ 튜브 공정은 중간공정 품목(TA)이 없어 테스트 불가. 튜브 대신 조립(AA) 또는 진공(VA) 중간공정 품목으로 대체. 대체 불가 시 스킵.
→ E15 김도영 (요청) — 또는 다른 부서 계정
  1. 입출고 → 창고 반출
  2. BOM 방식 / 중간공정 품목 (AA 또는 VA) 선택 / 수량 1 / 제출
→ E22 이필욱 (승인)
  1. 승인함 → 승인
A4 부서 → 창고 (창고 결재) ▼
→ E10 이지훈 (요청)
  1. 입출고 → 창고 반입
  2. 진공 부서 품목 (VA 또는 VF) / 낱개 / 수량 5 / 제출
→ E22 이필욱 (승인)
