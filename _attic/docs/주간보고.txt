창고를 박스 단위로 관리하는 기능을 새로 추가하고 생산 가능 수량 계산 기준을 정비하였으며, 김건호 과장님을 비롯한 현장 피드백을 반영해 입출고 동선을 개선하였습니다.

모바일 화면도 이제 현장에서 사용할 수 있는 수준으로 정비되었으니, 스마트폰으로도 한번 사용해 봐 주시면 감사하겠습니다.

■ 주요 진행 사항 (6/20 ~ 6/26)

1. 창고 지도 탭 개선 (6/22~6/26)
- 같은 품목이 여러 박스에 나뉘어 있을 때 박스별로 재고가 연동되어 줄어들도록 하는 기능을 추가하였습니다
- 박스를 끌어서 위치를 조정하고 수량을 입력하기 더 편하도록 개선하였습니다
- 권동환 사원님 요청을 반영하여, 파렛트와 통로에도 물건을 적재할 수 있도록 하였습니다

2. 생산 가능 수량 계산 기준 개편 (6/22~6/23)
- 모델마다 기준이 되는 최종 제품을 직접 지정할 수 있게 하여, 현장 기준에 맞는 생산 가능 수량이 나오도록 하였습니다

3. 입출고 입력 동선 개선 — 김건호 과장님 피드백 (6/22~6/24)
- 삭제한 품목을 입출고·대시보드에서 숨기고, 품목 검색 시 공백을 무시해 더 쉽게 찾도록 하였습니다
- 부족한 품목을 창고에서 바로 가져오는 흐름을 출고·출하의 분해 작업까지 넓혔고, 작성 중이던 입출고를 '이어서 하기'로 되살릴 수 있게 하였습니다
- 새 품목을 등록할 때 초기 재고를 부서·위치별로 나눠 입력할 수 있어, 등록 직후부터 위치별 재고가 잡힙니다

4. 요청 수정·확인 흐름 개선 (6/24)
- 이계숙 주임님 피드백을 반영하여, 승인 전 요청을 작성자가 직접 고칠 수 있게 하여 잘못 넣은 요청을 새로 만들지 않고 바로 수정합니다
- '내 요청' 목록에 더보기를 더해 본인 요청을 빠르게 확인하고 이어서 처리할 수 있습니다
- 김건호 과장님이 알려주신, 낱개 출고가 주간보고 집계에서 빠지던 문제를 바로잡아 주간보고 수치가 더 정확해졌습니다

5. 알림 정리 기능 추가 (6/25)
- 권동환 사원님 피드백을 반영하여, 알림을 하나씩 지우거나 읽은 알림을 한 번에 정리할 수 있게 하여 오래된 알림에 새 알림이 묻히지 않습니다

6. 로그인·접근 권한·집계 정확도 정비 (6/22~6/23)
- 로그인 시 직원 이름을 영문 자판으로 입력해도 한글로 자동 인식되도록 하여, 한·영 전환 없이 바로 검색됩니다
- 이름 초성만 입력해도 직원을 찾을 수 있게 하였습니다
- 입출고 화면 접근 권한을 로그인 계정과 연동해, 권한이 있는 사람만 들어가도록 정리하였습니다
- 대시보드 전체 수량에서 삭제 품목을 빼서 정상·부족·품절 합과 맞췄습니다

7. 모바일 화면 정비 — 현장 사용 가능 수준 도달 (6/22~6/25)
- 입출고·불량 등 주요 화면을 작은 화면에 맞게 다듬어, 이제 스마트폰에서도 현장 업무를 처리할 수 있는 수준으로 정비하였습니다

시스템은 제 자리 PC가 켜져 있을 때 http://192.168.0.63:3000 에서 확인하실 수 있습니다.
