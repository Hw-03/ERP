───────────────────────────────────────────────────────────

■ 주요 진행 사항 (6/13 ~ 6/19)

1. 입출고 실수 처리 방식 개선 — 정정에서 취소로 전환 (6/15)
- 기존 정정 방식은 단순 입출고에만 적용 가능하고, 생산·분해처럼 여러 부품의 재고를 동시에 움직이는 거래 유형은 정정할 방법이 없었습니다
- 이번에 취소 방식으로 전환하여, 원본 기록은 그대로 두고 재고 효과만 역방향으로 되돌리는 방식으로 모든 거래 유형에 동일하게 적용 가능
- 취소 이후에도 원본·취소 이력이 모두 남아 담당자가 언제 어떤 건을 왜 취소했는지 확인 가능

2. 생산 가능 수량 표시 기준 개선 중 (6/15~)
- 기존에도 대시보드에서 생산 가능 수량을 보여줬으나, 부품 단위로 표시되어 실제로 몇 대를 만들 수 있는지 바로 확인하기 어려웠습니다
- 최종 조립품(AF) 단위로 기준을 바꾸는 방향으로 구조 개편을 시작하였으며, 출하준비·빠른조립·총생산 가능 수량 3가지로 구분하는 설계를 적용하고 있습니다
- 현재는 초기 구조 변경 단계이며, 모델별 분리 표시·현장 기준 검증 등 아직 손볼 부분이 많이 남아 있어 계속 작업 중입니다

3. 인수인계 임시저장 및 수신 권한 개선 (6/15~6/16)
- 인수인계서 작성 중 임시저장 기능 추가, 작성 도중 이탈해도 내용이 보존되어 재입력 부담 감소
- 인수 확인 권한을 실제 물자를 넘겨받는 부서로 제한

4. 주간보고 화면 정확도 개선 (6/16)
- 생산 실적 집계에 입고 건이 포함되던 문제를 수정하고 전주 말 재고 기산을 정확히 하여 주간 변동 수치 신뢰성 향상
- 인쇄·CSV 출력 등 불필요 기능 제거로 화면 집중도 향상

5. 데스크톱 화면 개선 (6/15~6/17)
- 빠른 작업 진입 시 탭 튕김·품목 검색 깜빡임 수정으로 입출고 동선 안정화
- 창고 관리 탭을 구조 편집·위치 배정으로 통합하여 설정 접근 단계 감소

6. 모바일 하단 탭바 디자인 완성 (6/16)
- 선택한 탭 위치에 따라 부드럽게 이동하는 슬라이딩 pill 애니메이션 도입으로 탭 선택 시각 피드백 개선
- 탭 선택 시 눌림 효과 제거로 시각 잡음 감소, 렌더링 이탈 5건 수정으로 화면 안정성 확보

7. 모바일 화면 정비 진행 중 (6/17~)
- '더보기'를 바텀시트에서 정식 탭으로 전환하여 탐색 일관성 확보
- 불량 허브 진입 시 지표와 격리 목록을 첫 화면에 바로 표시, 이동 단계 감소
- 입출고 작성 중 하단 탭 이탈 시 저장 확인 시트 노출, 미입력 건 손실 방지
- 입출고 진행 바·헤더·섹션 탭·수량 버튼·드롭다운 등 전 단계 레이아웃을 모바일 화면에 최적화하는 작업을 진행 중이며, 순차적으로 개선하고 있습니다
