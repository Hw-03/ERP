### 품목 동일성 확인 — COCOON "4-2번 하네스" vs SOLO "5번 하네스"
#품목 #하네스 #데이터 #질문 #해결

COCOON의 "4-2번 하네스"와 SOLO의 "5번 하네스"가 동일한 품목이 맞는지 확인 요청.

→ 해결 (2026-06-16) — 사용자 판단으로 종결 처리 (실제 동일성 대조 결과는 별도 기록 없음)
- 현장 호칭("4-2번", "5번")이라 mes_code/품목명과 직접 매칭 안 됨 — 실제 품목 식별 먼저 필요.
- 확인 방법: COCOON·SOLO 모델의 하네스 품목 목록 조회 후 동일 품목(같은 mes_code) 여부 대조.
- 결과에 따라: 동일하면 한 품목 공유, 다르면 별도 품목 — 위 정렬/BOM 작업에도 영향.
- 개발 외 데이터 확인 건 — 담당자가 직접 대조.
