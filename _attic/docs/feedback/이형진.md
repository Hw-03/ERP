## 2. 빠른 작업칸 입고/출고 분리 + 최종 단계 바로 진입

> 입출고 진행 버튼을 입고/출고로 나눠, 눌렀을 때 작업 유형을 고르면 Step 3(대상 선택)에 바로 진입하면 좋겠다.

**상태: ✅ 구현 완료 (2026-06-12)**

- `InventoryDetailPanel.tsx` 단일 버튼 → `[입고]` / `[출고]` 2버튼으로 교체
- 버튼 클릭 시 인라인 팝업(포털 불필요):
  - 입고: 부서 입고 / 창고 반입 / 원자재 수령(창고 정·부 전용)
  - 출고: 부서 출고 / 창고 반출
- 선택 시 `IoEntryIntent` 생성 → `onGoToWarehouse(item, intent)` 콜백
- 입출고 위저드 마운트 시 1회 effect:
  - `setWorkType` → `setDeptIoDirection` 또는 `setSubType` → `goTo(3)` (Step 3 진입)
  - 기존 `useIoPreselect`가 다음 렌더에 품목 자동 카트 추가
- `canReceive` prop으로 원자재 수령 게이트 (비창고 사용자 미표시)
- 데스크톱·모바일 양쪽 동일하게 동작
