|---|---|---|
| `database` | `db` | `{"ok": bool}` 중첩 구조 |
| `tables` | `rows` | `{"items":N, "employees":N, "inventory":N, "transaction_logs":N, "queue_batches":N}` |
| `open_queue_count` | `open_queue_batches` | |
| `latest_transaction_at` | `last_transaction_at` | |
| (없음) | `status` | `"ok"` \| `"degraded"` — 문서에 미기재 |
| `inventory_mismatch_count` | `inventory_mismatch_count` | ✅ 일치 |

### 변경 후보 (docs/OPERATIONS.md:120-127)

```diff
- - `database`: `"ok"` 면 DB 연결 정상
+ - `db.ok`: `true` 면 DB 연결 정상
- - `tables`: 주요 테이블 행 수
+ - `rows`: 주요 테이블 행 수 (`items`, `employees`, `inventory`, `transaction_logs`, `queue_batches`)
- - `open_queue_count`: 미처리 큐 건 수
+ - `open_queue_batches`: 미처리 큐 배치 건 수
  - `inventory_mismatch_count`: Inventory 합계와 InventoryLocation 합계 불일치 건수 — `0` 이 정상
- - `latest_transaction_at`: 최근 거래 시각
+ - `last_transaction_at`: 최근 거래 시각 (ISO 8601, 없으면 null)
+ - `status`: `"ok"` (정상) \| `"degraded"` (DB 오류 또는 mismatch 존재)
```
