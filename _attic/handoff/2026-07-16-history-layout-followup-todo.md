## 완료 항목

1. 작업 열 알약을 최장 작업 라벨 기준 128px(`w-32`) 고정 폭으로 통일했다. 대표 행, BOM 묶음, 하위 품목, 재작업 결과 알약은 작업 열의 최대 폭 안에서 중앙 정렬과 긴 라벨 말줄임표를 유지한다.
2. 다품목 수량보정은 표시용 가상 묶음 행으로 2단계 접기·펼치기를 제공한다. 원시 `adjust_in`/`adjust_out`뿐 아니라 과거 수동 입고로 저장된 `produce` 배치도 같은 수량보정 입고로 해석한다. 입고는 `수량보정 입고`, 출고는 사용자 확정 라벨인 `출고`로 표시한다. 단건 보정에는 중간 묶음 행을 만들지 않는다.
3. 우측 상세 섹션명을 `실제 영향`에서 `재고 변화`로 통일했다. 다위치 변화는 위치·품목 수·순증감·chevron을 포함한 44px 이상 테두리 요약 행으로 표시한다.
4. 요청자·시각 영역은 시각을 한 줄로 우선 확보하고, 요청자명만 남은 폭에서 말줄임표 처리한다.
5. 재고 변화의 개별 품목 행을 읽기 전용 `div`로 바꿔 hover·focus·클릭과 목록 스크롤/강조를 제거했다. 위치 아코디언과 BOM 구성 목록 드릴다운은 유지한다.
6. 부서가 있는 단건 `DEFECT_SCRAP`은 `<부서> → 폐기` 흐름으로, 부서가 없는 레거시 기록은 기존 `불량 폐기` 단일 라벨로 표시한다.
7. 취소 확인 화면의 감사 문구 `되돌릴 실제 영향`은 범위 밖으로 유지했다.
