주말 동안 추가로 진행한 사항을 보고드립니다.

---

■ 진행 사항 (4/25~4/27)

**1. 입출고 화면 전면 개선**
품목 선택 → 수량 입력 → 확인 순서로 진행되는 단계형(wizard) 방식으로 재구성하였습니다.
각 단계를 순서대로 따라가면 되어 사용 중 실수가 발생하기 어렵게 설계되었습니다.
처리 중 일부 오류가 생기더라도 안내 팝업과 자동 새로고침으로 안전하게 복구됩니다.

**2. 코드 품질 대규모 정비**
시스템 내부 코드를 전면적으로 정비하였습니다.
- 데이터 조회 속도 최적화 (불필요한 중복 쿼리 제거)
- 오류 발생 시 응답 형식 전체 통일
- 테스트 자동화(CI) 도입 — 코드 수정 시 자동으로 오류를 감지
- 1,135줄짜리 대형 파일을 용도별로 분할하여 유지보수 편의성 향상

**3. BOM Where-Used 기능 추가**
특정 자재가 어떤 완제품의 BOM에 사용되는지 역방향으로 조회할 수 있는 기능을 추가하였습니다.

**4. ERP 카테고리 코드 수정**
내부 자재 분류 코드 중 `BA` 코드가 실제 운영 기준과 맞지 않아 `AA`로 전면 교체하였습니다.
현재는 개인 판단으로 수정한 사항이며, 추후 담당자 협의를 통해 확정이 필요합니다.

**5. 로그인 화면 최초 구현**
ERP 시스템 접속 시 로그인 화면이 표시되도록 최초로 구현하였습니다.
현재는 화면 구성(인트로 애니메이션 + 로그인 폼) 수준이며, 실제 권한 연동은 추후 진행 예정입니다.

**6. 버그 수정**
- 직원 목록에서 일부 인원이 비활성 처리되던 버그 수정
- 관리자 화면 확인 팝업을 시스템 기본 팝업에서 자체 UI로 교체
- 한글 인코딩 깨짐 현상 수정

**7. 비교 분석 문서 작성**
기존 MES 시스템과 현재 개발 중인 ERP의 차이점을 정리한 비교 분석 보고서를 작성하였습니다.
추후 내부 공유 또는 도입 검토 시 참고 자료로 활용 가능합니다.

화면은 제 자리 PC가 켜져 있을 때 http://192.168.0.63:3000 에서 확인하실 수 있습니다.

---

■ 추후 진행 예정

- **실제 데이터 입력** — 권동환 사원과 품목 명칭 통일 후 ERP에 순차 반영
- **로그인 및 권한 관리** — 부서·역할별 접근 권한 연동
- **카메라 QR·바코드 스캔** — 모바일 카메라 연동 완성
- **외부 접근 환경 구성** — 사내 PC 또는 NAS 서버 구축
