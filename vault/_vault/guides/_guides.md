## 쉬운 말로 설명

이 폴더는 **교과서** 역할이다. 실제 코드 파일(`*.py.md`, `*.tsx.md`)이 각 파일을 설명하는 것과 달리, 여기 있는 문서들은 **여러 파일에 걸친 흐름이나 개념**을 설명한다.

처음 코드를 공부할 때: `처음 읽는 사람` → `ERP MOC` → `용어 사전` → 시나리오 순으로 읽으면 된다.
