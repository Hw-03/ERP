## ⏱ 빠른 시연 동선 (6분 · 슬라이드 9 체크리스트와 1:1)

발표 중 alt+tab으로 MES 이동 → 아래 순서대로 → 슬라이드 복귀 시 `T`키로 체크.

| # | 슬라이드 항목 | MES 동작 | 메뉴 |
|---|---|---|---|
| 1 | PIN 로그인 | PIN 입력 → 대시보드 진입 | 로그인 |
| 2 | 창고 입출고 요청 | 입출고 → 요청 작성 → "창고 입출고" → 창고→부서 → 품목·수량 → 제출 | §2 |
| 3 | 창고 정/부 승인 처리 | 입출고 → "창고 승인함" → 요청 승인(PIN) | §2 |
| 4 | 부서 재고 현황 확인 | 대시보드 → 부서 필터 → 우측 품목 상세의 부서별 재고 | §1 |
| 5 | BOM 입고 → 하위 자동출고 | 입출고 → "부서 입출고" → 생산 → 완제품 입고 시 하위 자동 출고 확인 | §2 |
| 6 | 불량 등록 | 입출고 → "불량" → 불량 격리 → 제출 | §2 |

> 시연 계정: **조립 부서 + 생산부 정/부(이필욱·김건호)** 로 진행. (부서 결재가 같은-부서로 스코프돼 있어 타 부서 요청은 시연 막힘 — 발표 전 라우팅 수정 전제. 안 됐으면 조립으로만.)

## §1. 대시보드  `?tab=dashboard`

**들어가는 법**: 로그인 직후 기본 화면. 좌측 사이드바 "대시보드".
**화면 구성**: 상단 KPI 그룹 / 필터(접이식) / 검색 / 재고 테이블 / 행 클릭 시 우측 상세 패널.

**보여줄 기능 체크리스트**:
- [ ] KPI 그룹 클릭 — **전체 / 재고 많음 / 재고 적음 / 재고 없음** 전환 (클릭 시 자동 필터)
- [ ] "생산 가능 용량" 패널 → 클릭 시 부서별 용량 모달
- [ ] 필터 펼치기 — **부서**(다중 체크, "창고" 포함) / **모델 분류** / **공정 단계(R·A·F)**, "전체 해제"
- [ ] 검색창 — 품목명·품목코드로 실시간 필터
- [ ] 재고 테이블 행 클릭 → 우측 패널
- [ ] 우측 패널: 승인 대기 수량 / 사용 가능 재고 / **부서별 재고 위치(정상·불량 탭)** / 최근 거래 5건
- [ ] 우측 패널 "입출고 진행" → 이 품목 들고 입출고 탭으로 이동
- [ ] 패널 닫기 — X / ESC / 행 재클릭

**시연 멘트**: "엑셀 안 열어도 어느 부서에 몇 개 있는지 한 화면에서 보입니다. 빨강은 재고 부족."
**대표 흐름**: 재고 적음 KPI 클릭 → 품목 행 클릭 → 우측에서 부서별 분포 보여주기.
**주의**: 샘플 데이터 — 수치 자체보다 "한눈에 보인다"를 강조.
