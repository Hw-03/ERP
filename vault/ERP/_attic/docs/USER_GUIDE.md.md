# 사용자 가이드

DEXCOWIN MES 데스크톱 셸(`/legacy`)을 처음 사용하는 현장 사용자를 위한 화면별 사용법.

## 공통

- 좌측 상단 **DEXCOWIN** 로고 옆 사이드바로 화면을 전환한다(대시보드 · 입출고 · 입출고 내역 · 관리자).
- 우측 상단 **새로고침** 버튼은 현재 화면 데이터를 다시 불러온다.
- 우측 상단 **테마 토글**(달/해 아이콘)로 다크/라이트를 전환한다.
- 우측 상단 **상태 pill**:
  - 초록 점: "방금 완료 …" — 직전 작업이 정상 처리됨
  - 파란 점: 일반 상태 메시지
  - 빨간 점: 오류 또는 데이터를 불러오지 못함
  - 노란 점: 재고 부족·주의

빈 상태(데이터 없음 / 검색 결과 없음 / 필터로 가려짐)와 로드 실패는 어느 화면에서나 같은 시각 언어로 안내된다.

## 1. 대시보드 (재고)

전체 재고 현황을 빠르게 파악하고 필요 시 조치로 이어가는 화면.

KPI 카드 순서: **전체 → 정상 → 부족 → 품절** (이 순서는 고정).

- KPI 카드를 누르면 해당 상태의 품목만 필터링된다.
- 부서 칩 / 모델 칩 / 분류 칩으로 좁힐 수 있다. 모든 칩이 OFF면 "전체"와 같은 결과가 나온다.
- "조치 필요" 카드는 부족·품절 등 즉시 보충이 필요한 항목을 모아 보여준다. 행을 누르면 곧바로 입출고 화면에서 해당 품목으로 이동한다.
- 결과가 0건이면 "필터 초기화" 버튼으로 한 번에 풀 수 있다.

## 2. 입출고 (단계형 wizard)

작업은 **5단계** 카드 흐름으로 진행한다. 각 단계가 끝나면 다음 카드가 자동으로 열린다.

### 1단계 · 담당자

- 본인 카드를 누른다. 부서별 색상이 지정되어 있다.
- 직원이 10명을 넘으면 "추가 N명" 카드로 펼친다.

### 2단계 · 작업 유형

| 작업 유형 | 의미 |
|---|---|
| 원자재 입출고 | 창고 기준 입고/출고 |
| 창고 이동 | 창고↔생산부서 이동 |
| 부서 입출고 | 생산부서 기준 입고/출고 |
| 패키지 출고 | 등록된 묶음 출고 |
| 불량 등록 | 불량 격리 처리 (되돌릴 수 없음) |
| 공급업체 반품 | 공급업체 반품 처리 (되돌릴 수 없음) |

- 이동 방향, 대상 부서, 불량 발견 위치 등 작업에 필요한 옵션이 추가로 나타난다.
- 빨간 톤으로 표시되는 작업은 되돌릴 수 없는 작업이다. **이 작업으로 진행 →** 버튼을 눌러야 다음으로 넘어간다.

### 3단계 · 품목 선택

- 부서 / 모델 / 분류 / 검색으로 좁히고 품목 행을 클릭해 선택한다.
- 한 번에 100개씩 노출되며 **100개 더 보기** 버튼으로 더 불러올 수 있다.
