# 📁 .claude

## 이 폴더는 무엇을 위한 곳인가

Claude Code가 사용하는 로컬 작업 설정과 흔적입니다.

## 현장 업무와의 관계

시스템 기능 자체와는 직접 관계가 없습니다.

## 언제 보면 좋나

- AI 작업 환경 설정을 확인할 때

## 어떤 의미로 봐야 하나

이 폴더는 MES 기능을 만드는 코드가 아니라, AI 도구가 작업을 돕기 위해 남기는 로컬 설정에 가깝습니다. 따라서 재고, 입출고, 직원, 관리자 기능을 이해하려고 이 폴더를 먼저 볼 필요는 없습니다.

작업 중 AI가 어떤 설정으로 움직이는지, 특정 로컬 도구가 왜 실행되는지 확인할 때만 참고하면 됩니다.

## 보통 보지 않아도 되는 이유

- 실제 사용자 화면과 직접 연결되지 않습니다.
- DB 구조나 API 계약을 정의하지 않습니다.
- 개인 작업 환경에 가까운 내용이 섞일 수 있습니다.
- main/vault-sync 정책을 이해할 때도 실제 기준은 Git 브랜치와 `vault/` 정책입니다.

## 조심할 점

로컬 도구 설정이므로 프로젝트 기능 문서처럼 다루지 않습니다.
