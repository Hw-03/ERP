# 📁 legacy

## 이 폴더는 무엇을 위한 곳인가

현재 운영 중인 실제 MES 화면입니다. 이름은 legacy지만 지금 사용자가 보는 핵심 UI입니다.

## 현장 업무와의 관계

대시보드, 입출고, 내역, 관리자, 로그인 게이트가 이 흐름 안에서 움직입니다.

## 언제 보면 좋나

- 현장 UX를 바꿀 때
- 로그인 담당자 흐름을 볼 때
- 데스크톱/모바일 화면 분기를 확인할 때
