# 📁 images

## 이 폴더는 무엇을 위한 곳인가

`frontend/public/images`는 프론트엔드 화면이나 공용 로직의 세부 폴더입니다.

## 현장 업무와의 관계

사용자가 보는 화면이나 화면이 서버와 통신하는 방식에 연결됩니다.

## 언제 보면 좋나

- 이 폴더 안의 파일이 어떤 역할인지 빠르게 파악할 때
- 수정 전에 먼저 읽을 파일을 고를 때

## 주요 하위 폴더

- [[ERP/frontend/public/images/items/📁_items]] — 품목 코드별 사진 자산입니다. 파일 수가 많아 개별 설명을 만들지 않고 이 폴더 설명으로 묶어 관리합니다.
- [[ERP/frontend/public/images/login/📁_login]] — `frontend/public/images/login`는 프론트엔드 화면이나 공용 로직의 세부 폴더입니다.

## 조심할 점

폴더 성격을 먼저 확인하고 현재 운영 코드인지, 보관 자료인지, 자동 생성물인지 구분해야 합니다.
