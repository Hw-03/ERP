# 📁 items

## 이 폴더는 무엇을 위한 곳인가

품목 코드별 사진 자산입니다. 파일 수가 많아 개별 설명을 만들지 않고 이 폴더 설명으로 묶어 관리합니다.

## 현장 업무와의 관계

현장에서 품목을 눈으로 구분할 때 쓰이는 이미지 원본입니다.

## 언제 보면 좋나

- 품목 사진이 안 보일 때
- 특정 품목 코드의 이미지가 있는지 확인할 때

## 파일 설명 방식

이 폴더는 자동 생성물이나 반복 자산 성격이 강해서 개별 파일 설명을 만들지 않습니다. 폴더의 존재 이유만 이해하면 충분합니다.

## 조심할 점

파일명이 품목 코드와 연결되므로 이름을 바꾸면 화면에서 사진을 찾지 못할 수 있습니다.
