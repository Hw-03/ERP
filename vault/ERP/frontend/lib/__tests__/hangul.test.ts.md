# hangul.test.ts

## 이 파일은 뭐예요?
`toChosung` 함수가 한글 완성형 음절에서 초성 자모를 추출하는지 검증하는 단위 테스트입니다. 초성 검색(예: `"ㄱㄱㅎ"`으로 `"김건호"` 찾기) 기능의 기반 유틸리티를 커버합니다.

## 언제 보나요?
- `frontend/lib/hangul.ts`의 `toChosung` 구현을 수정할 때
- 초성 검색이 특정 문자에서 오작동할 때

## 중요한 내용
- 완성형 음절 → 초성 변환: `"김건호"` → `"ㄱㄱㅎ"`
- 겹받침(닭·값 등) 무관하게 초성만 추출
- 호환 자모(`ㄱㄱㅎ`)는 그대로 통과
- 라틴자·숫자·공백 등 비한글 문자는 변경 없이 통과
- 빈 문자열 입력 → 빈 문자열 반환
