# 📁 msw

## 이 폴더는 뭐예요?
프론트엔드 테스트에서 백엔드 API를 가짜로 흉내 내주는 MSW(Mock Service Worker) 설정 모음입니다. `server.ts`가 모든 핸들러를 합쳐 테스트용 가짜 서버를 만들고, `handlers/` 하위 폴더에 도메인별 응답 규칙이 담겨 있습니다.

## 언제 여기를 보나요?
- 테스트를 돌렸는데 API 응답이 예상과 달라서 모킹 로직을 확인할 때
- 새 API 엔드포인트를 추가한 뒤 테스트용 핸들러도 만들어야 할 때
- 테스트 환경에서 특정 API가 호출되지 않거나 404가 뜰 때

## 주요 파일
- `server.ts` — 모든 핸들러를 조합해 MSW 서버 인스턴스를 생성·내보냄
