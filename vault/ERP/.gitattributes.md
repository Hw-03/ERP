# .gitattributes — .gitattributes 설명

## 이 파일은 무엇을 책임지나

`.gitattributes`는 Git/도구 설정입니다. 프로젝트 구조 안에서 `.gitattributes` 위치에 있으며, 필요할 때 역할과 연결 파일을 확인하기 위한 설명을 둡니다.

## 업무 흐름에서의 의미

프로젝트 운영과 개발을 이해하기 위한 보조 정보입니다.

## 언제 보면 좋나

- 이 파일이 맡은 화면/API/데이터 흐름을 확인해야 할 때
- 수정 전에 영향 범위를 빠르게 파악해야 할 때

## 중요한 내용

자동으로 뽑을 수 있는 함수/클래스 목록은 적지만, 파일 위치와 확장자로 볼 때 위 역할을 맡습니다.

## 연결되는 파일

- [[ERP/📁_ERP]] — 이 파일이 속한 폴더의 안내판입니다.

## 핵심 발췌

```text
# Windows-only scripts: force CRLF line endings on checkout AND in GitHub ZIP archives.
# Without this, GitHub "Download ZIP" serves these files with LF endings, which makes
# cmd.exe misparse the batch syntax (commands split mid-line, fragments treated as
# separate commands). Affects anyone who downloads via ZIP instead of git clone.
*.bat   text eol=crlf
*.cmd   text eol=crlf
*.ps1   text eol=crlf

# Shell scripts: force LF (in case anyone commits one from Windows in the future).
*.sh    text eol=lf
```
