# 📁 baselines

## 이 폴더는 무엇을 위한 곳인가

`_dev/baselines`는 현재 운영 코드가 아니라 보관/참고 자료입니다.

## 현장 업무와의 관계

과거 작업 맥락을 볼 때 도움이 되지만 현재 기준으로 바로 쓰면 안 됩니다.

## 언제 보면 좋나

- 이 폴더 안의 파일이 어떤 역할인지 빠르게 파악할 때
- 수정 전에 먼저 읽을 파일을 고를 때

## 먼저 볼 파일 5개

- [[ERP/_dev/baselines/openapi.json]] — `openapi.json`는 JSON 설정/데이터입니다. 프로젝트 구조 안에서 `_dev/baselines/openapi.json` 위치에 있으며, 필요할 때 역할과 연결 파일을 확인하기 위한 설명을 둡니다.

## 조심할 점

폴더 성격을 먼저 확인하고 현재 운영 코드인지, 보관 자료인지, 자동 생성물인지 구분해야 합니다.
