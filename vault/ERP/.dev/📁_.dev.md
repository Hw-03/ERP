# 📁 .dev

## 이 폴더는 무엇을 위한 곳인가

`.dev`는 개발 도구가 쓰는 설정 또는 산출물 폴더입니다.

## 현장 업무와의 관계

제품 기능보다 개발/검증 환경에 가깝습니다.

## 언제 보면 좋나

- 이 폴더 안의 파일이 어떤 역할인지 빠르게 파악할 때
- 수정 전에 먼저 읽을 파일을 고를 때

## 파일 설명 방식

이 폴더는 자동 생성물이나 반복 자산 성격이 강해서 개별 파일 설명을 만들지 않습니다. 폴더의 존재 이유만 이해하면 충분합니다.

## 조심할 점

폴더 성격을 먼저 확인하고 현재 운영 코드인지, 보관 자료인지, 자동 생성물인지 구분해야 합니다.
